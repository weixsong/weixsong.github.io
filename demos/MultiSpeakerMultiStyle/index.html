Authors: Wei Song, Guanghui Xu, Zhengchen Zhang, Chao Zhang, Xiaodong He, Bowen Zhou

Abstract: Neural vocoder, such as WaveGlow, has become an important component in recent high-quality text-to-speech (TTS) systems. In this paper, we propose Efficient WaveGlow (EWG), which is a flow-based generative model serving as an efficient neural vocoder. Similar to WaveGlow, Efficient WaveGlow has a normalizing flow backbone where each flow step consists of an affine coupling layer and an invertible 1Ã—1 convolution. To reduce the model parameters and enhance the speed without sacrificing the quality of the synthesized speech, Efficient WaveGlow improves WaveGlow in the following aspects. First, the WaveNet-style transformation network in WaveGlow is replaced with an FFTNet-style dilated convolution network. Next, to reduce the computation cost, group convolution is applied to both audio and local condition features. At last, the local condition is shared among the transformation network layers in each coupling layer. As a result, Efficient WaveGlow can reduce the number of floating-point operations (FLOPs) required to generate one-second audio and the number of model parameters both by more than 12 times. Experimental results show that Efficient WaveGlow can reduce real-world inference time cost by more than twice, without any obvious reduction in the quality of the synthesized speech.
